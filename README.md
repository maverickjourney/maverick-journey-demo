# Maverick Journey Prototype V5

Full trainer + client demo prototype.

## Includes
### Trainer side
- Coach OS
- 10-client roster
- Client profile
- AI assistant
- Slack command layer
- Reports
- Reminders
- Tools/Admin
- Pitch flow

### Client side
- Client login-style dashboard
- Daily logging
- Nutrition dashboard
- Meal photo examples
- Progress photo timeline
- Client messages
- Client reminders
- Weekly client report

## Vercel Settings
Framework: Vite
Build command: npm run build
Output directory: dist
Install command: npm install
