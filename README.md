# Maverick Journey Prototype

Static Vercel-ready prototype for a Maverick Journey AI coaching command center.

## Deploy to Vercel
1. Create a new GitHub repo.
2. Upload these files.
3. Go to Vercel > Add New Project > Import repo.
4. Framework preset: Vite or Other.
5. Build command: `npm run build`
6. Output directory: `dist`

## Local preview
```bash
npm install
npm run start
```
