# Maverick Journey Routed Prototype V6

This version creates 3 separate URLs on the same Vercel project:

- `/` = main trainer dashboard
- `/slack/` = Slack trainer/client workflow demo
- `/client/` = client dashboard/login experience

## Vercel Settings
Build command: npm run build
Output directory: dist
Install command: npm install
